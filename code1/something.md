  # 你以后画图可以按这个思路选

  我给你一个特别实用的“小白三问法”。

  每次画图前，先问自己：

  ### 1. 我现在是在看“一个变量”还是“多个变量”？

  - 一个变量：先想直方图、箱线图
  - 两个变量：先想散点图
  - 多组比较：先想柱状图
  - 时间变化：先想折线图

  ———

  ### 2. 我想看的是“分布、对比、关系、趋势”中的哪一种？

  - 分布 -> 直方图
  - 对比 -> 柱状图
  - 关系 -> 散点图
  - 趋势 -> 折线图

  ———

  ### 3. 这张图画出来后，我希望得出一句什么结论？

  比如你这里希望得到的结论是：

  - SalePrice 分布右偏
  - 存在高价长尾
  - 所以后续考虑做对数变换

  如果你说不出这句话，说明图画得还不够有目的

### 4. 选图速查表
看单个数值变量分布：hist
  - 看异常值和离散程度：boxplot
  - 看类别数量对比：bar
  - 看两个数值变量关系：scatter
  - 看训练过程变化：plot
  - 看相关性矩阵：heatmap
